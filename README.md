## Installing requirements

```sh
# Install writ-cms
git clone git@github.com:scriptype/writ-cms.git
cd writ-cms
npm i

# Install 'writ' binary
npm i -g .
```

## Start developing and live editing

```sh
writ start
```

## Build for deployment

```sh
writ build
```
